# Découvrez les méthodes factorielles et la classification non supervisée

Parmis les méthodes d'apprentissage non supervisé on a :
- les méthodes factorielles

- les méthodes de classification non supervisé aussi appelé partionnement de données ou clustering.

## Méthodes factorielles
Parmis les méthodes factorielles la plus célèbre est l'ACP (Analyse par composante principale) ou PCA(Principal Component Analysis) en Anglais .

### Intérêt de l'ACP
L'ACP permet de réduire le nombre de variables en trouvant de nouvelles variables qui en synthétisent plusieurs.
Remplacer plusieurs variables du tableau par une variable synthetique facilite l'analyse mais fait perdre de l'information.

### Grille d'analyse de l'ACP

L'ACP permet d'etudier :

- La variabilité des individus : c'est à dire les ressemblances et les différences entre les individus.

- Les liaisons entre les variables : c'est à dire si y a des variables très corrélés pouvant être regroupés en variables synthétiques


## Méthodes de classification non supervisé
Concernant les méthodes de classification non supervisé on a l'algorithme de k-means(**k-moyennes**).

### Intérêt du clustering
Le clustering permet de regrouper les individus similaires , c'est à dire partitionner l'ensemble des individus .

### Application du clustering

- marketing : segmenter une base de données client et étudier leurs caractéristiques.

- analyse d'image : regroupement de pixels similaires en termes de couleurs en une seule couleur .

- classification des espèces animales.


**Note** : En Anglais on dit `classification` pour classification supervisée et `clustering` pour classification non supervisé
